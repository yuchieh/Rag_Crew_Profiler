# Step 4: 嚴格分離的 Agent 定義 (`agents.yaml`)

根據我們的開發規範，應避免把職責和邏輯直接寫死在 Python 檔案中。我們在 `config/agents.yaml` 中定義了三個分工明確的 AI 角色：

1. **User Profiler (使用者輪廓分析師)**: 負責歸納這名消費者的喜好（偏好什麼口味、平常打分嚴不嚴格）。
2. **Item Analyst (店家分析師)**: 負責萃取這間餐廳的特色（有什麼招牌菜、別人抱怨過什麼）。
3. **Prediction Modeler (評論預測家)**: 大腦指揮官，拿著前兩者的報告進行綜合判斷，最終進行模擬產出。
